Query can be delayed and/or executed with an interval.

Query for this chart has a delay of 10 secs.

([] time:(00:00:01*til 50)+.z.Z; price:50+sums -0.5+50?2)

Query for this chart has a delay of 10 secs and interval 5 secs.

([] time:(00:00:01*til 50)+.z.Z; price:50+sums -0.5+50?2)

Special $i$ reference can be used to know the number of iterations.

enlist[`processed]!enlist (3*$i$) mod 100
